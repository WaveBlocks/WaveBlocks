class WaveBlocks.HagedornPropagator(potential, packet, leading_component, parameters)¶

This class can numerically propagate given initial values in a potential . The propagation is done for a given set of homogeneous Hagedorn wavepackets neglecting interaction.

Return the wavepackets taking part in the simulation.

Parameters:packet (Integer) – The number of a single packet that is to be returned.

number_components = None¶

Number of components the wavepacket has got.

packets = None¶

A list of Hagedorn wavepackets together with some codata like the leading component which is the index of the eigenvalue of the potential that is responsible for propagating the Hagedorn parameters.

potential = None¶

The potential the packet(s) feel.

propagate()¶

Given the wavepacket at time compute the propagated wavepacket at time . We perform exactly one timestep here.
