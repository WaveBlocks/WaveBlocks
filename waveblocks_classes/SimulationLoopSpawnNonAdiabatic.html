Class documentation¶

class WaveBlocks.SimulationLoopSpawnNonAdiabatic(parameters)¶

This class acts as the main simulation loop. It owns a propagator that propagates a set of initial values during a time evolution. All values are read from the I{Parameters.py} file.

end_simulation()¶

Do the necessary cleanup after a simulation. For example request the IOManager to write the data and close the output files.

iom = None¶

A I{IOManager} instance for saving simulation results.

prepare_simulation()¶

Set up a Spawning propagator for the simulation loop. Set the potential and initial values according to the configuration. @raise ValueError: For invalid or missing input data.

propagator = None¶

The time propagator instance driving the simulation.

run_simulation()¶

Run the simulation loop for a number of time steps. The number of steps is calculated in the I{initialize} function.
