Class FourierPropagator

source code

Propagator'.Propagator --+
                         |
                        FourierPropagator

This class can numerically propagate given initial values $\Ket{\Psi}$ in a potential surface $V\ofs{x}$. The propagation is done with a Strang splitting of the time propagation operator.

Initialize a new FourierPropagator instance. Precalculate also the grid and the propagation operators.

  • potential - The potential the state $\Ket{\Psi}$ feels during the time propagation.
  • initial_values - The initial values $\Ket{\Psi\ofs{t=0}}$ given in the canonical basis.
Raises:
  • ValueError - If the number of components of $\Ket{\Psi}$ does not match the number of energy levels $\lambda_i$ of the potential.
