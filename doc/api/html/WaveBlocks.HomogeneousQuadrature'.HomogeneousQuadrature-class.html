quadrature(self, packet, operator=None, summed=False, component=None, diag_component=None)

source code 

Performs the quadrature of $\Braket{\Psi|f|\Psi}$ for a general $f$.

Parameters:
  • packet - The wavepacket $|\Psi>$.
  • operator - A real-valued function $f(x):R ightarrow R^{N imes N}.$
  • summed - Whether to sum up the individual integrals $\Braket{\Phi_i|f_{i,j}|\Phi_j}$.
  • component - Request only the i-th component of the result. Remember that $i \in [0, N^2-1]$.
  • diag_component - Request only the i-th component from the diagonal entries, here $i \in [0, N-1]$
Returns:
The value of $\Braket{\Psi|f|\Psi}$. This is either a scalar value or a list of $N^2$ scalar elements.
Raises:
  • NotImplementedError - Abstract interface.
Overrides: Quadrature'.Quadrature.quadrature

Note: 'component' takes precedence over 'diag_component' if both are supplied. (Which is discouraged)
