Initialize the new spawn condition. Most subclasses do not have to change this method. But some may need to set up persistent data structures e.g. for keeping histories of some values. This should be done here.

:param parameters: The simulation parameters. :type parameters: A :py:class:`ParameterProvider` instance. :param env: The caller environment. :type env: A :py:class:`Propagator` subclass instance.
