Class HagedornMultiPropagator

source code

Propagator'.Propagator --+
                         |
                        HagedornMultiPropagator

This class can numerically propagate given initial values $\Ket{\Psi}$ in a potential $V\ofs{x}$. The propagation is done for a given inhomogeneous Hagedorn wavepacket.
