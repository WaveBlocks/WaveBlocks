Class SpawnSimplePropagator

source code

Propagator.Propagator --+
                        |
                       SpawnSimplePropagator

This class can numerically propagate given initial values $\Ket{\Psi}$ in a potential $V\ofs{x}$. The propagation is done for a given homogeneous Hagedorn wavepacket.
