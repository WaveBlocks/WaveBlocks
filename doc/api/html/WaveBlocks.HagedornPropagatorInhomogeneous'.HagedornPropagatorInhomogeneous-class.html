Class HagedornPropagatorInhomogeneous

source code

Propagator'.Propagator --+
                         |
                        HagedornPropagatorInhomogeneous

This class can numerically propagate given initial values :math:`\Psi` in a potential :math:`V(x)`. The propagation is done for a given set of inhomogeneous Hagedorn wavepackets neglecting interaction.
