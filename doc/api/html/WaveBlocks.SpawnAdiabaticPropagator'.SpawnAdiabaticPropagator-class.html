Class SpawnAdiabaticPropagator

source code

Propagator'.Propagator --+
                         |
                        SpawnAdiabaticPropagator

This class can numerically propagate given initial values :math:`\Psi` in a potential :math:`V(x)`. The propagation is done for several given homogeneous Hagedorn wavepackets neglecting interaction.
