Module surfcf

The WaveBlocks Project

Function for plotting functions of the type f:I^2 -> C with abs(f) as y-value and phase(f) as color code. This function makes a 3D surface plot.

surfcf(gridx, gridy, phase, modulus, colormap=None)

source code 

Plot the modulus of a complex valued function $f:R^2 -> C$ together with its phase in a color coded fashion.

Parameters:
  • gridx - The grid nodes along the x axis of the real domain R^2
  • gridy - The grid nodes along the y axis of the real domain R^2
  • phase - The phase of the complex domain result f(grid)
  • modulus - The modulus of the complex domain result f(grid)
  • colormap - The colormap to use, if none is given, compute the 'default' QM colormap.
